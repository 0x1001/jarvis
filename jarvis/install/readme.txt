Prerequisites for Jarvis:
	- Python 2.7 (Recommended: https://code.google.com/p/pythonxy/)
	- Pybrain - http://pybrain.org/
	- pyttsx - http://pyttsx.readthedocs.org/en/latest/ (Older version https://pypi.python.org/pypi/pyttsx/1.0)

Installation:
	- git clone https://github.com/0x1001/jarvis.git
	- start.py - Starts Jarvis
	- test/test.py - Starts Jarvis tests
