Prerequisites for Jarvis:
	- Python 2.7 (Recommended for Windows: https://code.google.com/p/pythonxy/)
		For Linux:
			- apt-get install python-setuptools
			- apt-get install python-pip
			- apt-get install python-numpy
			- apt-get install python-scipy

	- Pybrain - http://pybrain.org/
		- git clone git://github.com/pybrain/pybrain.git
		- python setup.py install

    - pyttsx - http://pyttsx.readthedocs.org/en/latest/ (Older version https://pypi.python.org/pypi/pyttsx/1.0)
		- pip install pyttsx

    - mplayer.py - https://pypi.python.org/pypi/mplayer.py
        - python setup.py install
		- sudo apt-get install mplayer2

Installation:
	- git clone https://github.com/0x1001/jarvis.git
	- start.py - Starts Jarvis
	- test/test.py - Starts Jarvis tests
